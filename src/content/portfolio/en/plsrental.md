**PLS Rental** is a leading provider of technical equipment rental and production services for events. Focusing on premium sound systems, professional lighting, and seamless technical production, they cater to corporate events, government institutions, and large-scale productions across Indonesia, especially in the East Java region.

## Project Overview

This project involved designing and developing the official corporate website for PLS Rental. The objective was to create a digital platform that effectively communicates their professionalism, showcases their extensive service catalog, and provides a clear point of contact for prospective clients looking for high-quality event production services.

## Key Features

- 🎯 **Service Showcase** — Dedicated sections highlighting their core competencies in premium audio, lighting, and event technical support.
- 🛠️ **Custom CMS** — Built a bespoke Content Management System tailored specifically for their operational needs, allowing the team to easily manage services and content without technical expertise.
- 📈 **SEO Optimization** — Implemented comprehensive technical SEO, structural data (JSON-LD), and optimized metadata to ensure high search engine visibility for event rental queries.
- 📱 **Fully Responsive Design** — Optimized for both desktop and mobile viewing, providing a seamless browsing experience for clients on the go.
- ⚡ **High Performance** — Built with modern web technologies to ensure lightning-fast load times and smooth navigation.

## Technical Details

The website is engineered for speed and maintainability, utilizing a modern tech stack to deliver a fast, reliable, and SEO-friendly user experience that converts visitors into leads.
